# Pesquisa de Satisfação - TudoWeb

Programa em Python para coleta de opinião de clientes sobre o atendimento.

## Requisitos atendidos:
- Uso de estrutura de repetição `for`
- Coleta de nome, idade e opinião (1=EXCELENTE, 2=BOM, 3=RUIM)
- Pesquisa com 50 entrevistados
- Contagem de respostas EXCELENTE e RUIM
- Uso de estruturas de decisão (if/elif)
- Validação básica de entrada

## Como executar:
```bash
python pesquisa_tudoweb.py
